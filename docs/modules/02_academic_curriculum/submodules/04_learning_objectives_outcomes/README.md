The Learning Objectives & Outcomes submodule defines measurable learning objectives and outcomes for each topic, maps to Bloom's taxonomy, establishes success criteria, tracks student achievement against defined outcomes, and ensures outcome-based education aligned with NEP 2020 and board requirements.

### Purpose

Establish clear, measurable SMART learning objectives for all topics, define success criteria, map to Bloom's taxonomy levels, enable outcome-based assessment, track achievement rates, and ensure alignment with curriculum standards.

### Scope

- **SMART Objectives**: Specific, Measurable, Achievable, Relevant, Time-bound
- **Bloom's Taxonomy**: Cognitive level mapping
- **Success Criteria**: Achievement thresholds
- **Outcome Tracking**: Student achievement monitoring
- **Assessment Alignment**: Outcome-based evaluation

## üìä DATABASE SCHEMA

### Learning Outcomes Table

```sql
CREATE TABLE learning_outcomes (
  outcome_id INT PRIMARY KEY AUTO_INCREMENT,
  
  -- Curriculum Link
  subject_id INT NOT NULL,
  chapter_id INT,
  topic VARCHAR(300) NOT NULL,
  grade VARCHAR(10) NOT NULL,
  
  -- Objective Details
  objective_statement TEXT NOT NULL,
  objective_type ENUM('KNOWLEDGE', 'SKILL', 'ATTITUDE') NOT NULL,
  
  -- SMART Criteria
  is_specific BOOLEAN DEFAULT TRUE,
  is_measurable BOOLEAN DEFAULT TRUE,
  is_achievable BOOLEAN DEFAULT TRUE,
  is_relevant BOOLEAN DEFAULT TRUE,
  is_timebound BOOLEAN DEFAULT TRUE,
  
  -- Bloom's Taxonomy
  blooms_level ENUM(
    'REMEMBER', 'UNDERSTAND', 'APPLY',
    'ANALYZE', 'EVALUATE', 'CREATE'
  ) NOT NULL,
  
  -- Assessment
  assessment_method VARCHAR(500),
  success_criteria TEXT,
  target_achievement_percentage DECIMAL(5,2) NOT NULL,
  
  -- Marks
  marks_allocated INT,
  
  -- Competency Mapping
  competencies_mapped JSON,
  
  -- Status
  outcome_status ENUM('DRAFT', 'APPROVED', 'ACTIVE', 'ARCHIVED') DEFAULT 'DRAFT',
  
  -- Metadata
  created_date TIMESTAMP DEFAULT CURRENT_TIMESTAMP,
  
  -- Foreign Keys
  FOREIGN KEY (subject_id) REFERENCES subjects(subject_id),
  
  -- Indexes
  INDEX idx_subject_grade (subject_id, grade),
  INDEX idx_blooms (blooms_level),
  INDEX idx_status (outcome_status)
);
```

### Outcome Achievement Table

```sql
CREATE TABLE outcome_achievement (
  achievement_id INT PRIMARY KEY AUTO_INCREMENT,
  
  -- Student & Outcome
  student_id INT NOT NULL,
  outcome_id INT NOT NULL,
  
  -- Academic Year
  academic_year VARCHAR(9) NOT NULL,
  
  -- Achievement
  achieved BOOLEAN DEFAULT FALSE,
  achievement_score DECIMAL(5,2),
  achievement_date DATE,
  
  -- Assessment
  assessment_id INT,
  
  -- Metadata
  created_date TIMESTAMP DEFAULT CURRENT_TIMESTAMP,
  
  -- Foreign Keys
  FOREIGN KEY (student_id) REFERENCES students(student_id) ON DELETE CASCADE,
  FOREIGN KEY (outcome_id) REFERENCES learning_outcomes(outcome_id),
  
  -- Indexes
  INDEX idx_student_outcome (student_id, outcome_id),
  INDEX idx_achieved (achieved),
  INDEX idx_academic_year (academic_year)
);
```

## ‚öôÔ∏è BUSINESS RULES

### Rule 1: Outcome Achievement Calculation

```javascript
FUNCTION calculate_outcome_achievement(outcome_id, academic_year) {
  // Get total students
  total_students = QUERY("
    SELECT COUNT(DISTINCT student_id) as count
    FROM outcome_achievement
    WHERE outcome_id = ?
    AND academic_year = ?
  ", [outcome_id, academic_year])
  
  // Get students who achieved outcome
  achieved_students = QUERY("
    SELECT COUNT(*) as count
    FROM outcome_achievement
    WHERE outcome_id = ?
    AND academic_year = ?
    AND achieved = TRUE
  ", [outcome_id, academic_year])
  
  // Calculate achievement percentage
  achievement_percentage = (achieved_students.count / total_students.count) * 100
  
  // Get target
  outcome = GET_OUTCOME(outcome_id)
  target = outcome.target_achievement_percentage
  
  // Compare with target
  gap = achievement_percentage - target
  
  IF gap < -5 {
    // More than 5% below target
    SEND_ALERT("curriculum_team", {
      title: "Learning Outcome Below Target",
      message: `Outcome ${outcome_id}: ${achievement_percentage}% achieved (Target: ${target}%)`,
      gap: gap,
      priority: "MEDIUM"
    })
  }
  
  RETURN {
    total_students: total_students.count,
    achieved_students: achieved_students.count,
    achievement_percentage: achievement_percentage,
    target: target,
    gap: gap,
    status: gap >= 0 ? "MET" : "BELOW_TARGET"
  }
}
```
