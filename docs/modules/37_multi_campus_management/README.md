## DETAILED FINANCIAL CONSOLIDATION

### Month-End Close Process

**Timeline:** 1st to 5th of every month (for previous month)

**Day 1-2 (Campus Level):**
- Each campus closes books for the month
- Reconcile bank accounts, fee collections
- Record expenses, payroll
- Generate campus-level P&L, Balance Sheet

**Day 3 (Group Finance Team):**
- Collect financial data from all 4 campuses
- Validate data completeness, accuracy
- Identify discrepancies, request corrections

**Day 4 (Consolidation):**
- Consolidate revenue, expenses across campuses
- Eliminate inter-campus transactions (if any)
- Calculate group-level P&L, Balance Sheet
- Prepare variance analysis (actual vs budget)

**Day 5 (Reporting):**
- Finalize consolidated financial statements
- Prepare management reports, dashboards
- Present to Group CFO, CEO
- Share with campus principals

### 2. TO STUDENT MANAGEMENT (ALL CAMPUSES)

**WHY This Connection Exists:**
Inter-campus student transfers require coordination between Student Management modules across campuses.

**DATA FLOW:**
- **Transfer Request:**
  - Student ID, current campus, target campus
  - Reason for transfer, parent request
  - Academic records, fee status
- **Transfer Approval:**
  - Seat availability at target campus
  - Principal approval (both campuses)
  - Fee adjustment (if applicable)
- **Transfer Execution:**
  - Student record moved to target campus
  - TC (Transfer Certificate) issued by source campus
  - Admission at target campus

**TRIGGER EVENT:**
- **Transfer Request Submitted:** Parent/student requests transfer
- **Approval Granted:** Both principals approve
- **Transfer Completed:** Student record updated in both systems

**BUSINESS LOGIC:**
```
FUNCTION ProcessInterCampusTransfer(studentID, sourceCampus, targetCampus):
    // Step 1: Validate request
    student = GetStudentRecord(studentID, sourceCampus)
    IF student.feesDue > 0:
        RETURN "Transfer blocked: Outstanding fees"
    
    // Step 2: Check seat availability
    targetSeats = GetAvailableSeats(targetCampus, student.grade)
    IF targetSeats == 0:
        RETURN "Transfer blocked: No seats available"
    
    // Step 3: Get approvals
    sourceApproval = GetPrincipalApproval(sourceCampus, studentID)
    targetApproval = GetPrincipalApproval(targetCampus, studentID)
    IF NOT (sourceApproval AND targetApproval):
        RETURN "Transfer blocked: Approval pending"
    
    // Step 4: Execute transfer
    IssueTC(studentID, sourceCampus)
    MoveStudentRecord(studentID, sourceCampus, targetCampus)
    AdmitStudent(studentID, targetCampus)
    NotifyParents(studentID, "Transfer completed")
    
    RETURN "Transfer successful"
```

**EXAMPLE:**
- **Student:** Rohan Sharma (Grade 8, Main Campus)
- **Request:** Transfer to North Campus (family relocated to Gurgaon)
- **Date:** 15-Jun-2025
- **Process:**
  - Parent submits transfer request (15-Jun)
  - Fee clearance verified (no dues)
  - Seat availability confirmed at North Campus (Grade 8: 5 seats available)
  - Main Campus principal approves (17-Jun)
  - North Campus principal approves (18-Jun)
  - TC issued by Main Campus (20-Jun)
  - Admission at North Campus (22-Jun)
  - Classes start at North Campus (25-Jun)
- **Result:** Seamless transfer completed in 10 days
